## 界面设置

>  关闭不必要的工具栏，侧边栏，面包屑导航等

### 底部状态栏 + 侧边工具栏 ＋ 顶部工具栏 + 导航栏

均可以在 Views 选项中关闭

Views > Statu Bar

Views > Tool Buttons

Views > Tool Bar 

Views > Navigation Bar 

### 结构层次面包屑导航

鼠标移动至 HTML 或 CSS 中某一行时，在文件上方会出现光标目前位置所处的层次，平时不怎么用。

Preferences > **search** "breadcrumbs" > **close** 'Show breadcrumbs'

### Tabs 栏

可以在任意一个 Tab 上右键: 

Tabs Placement > none
